# Model Metrics
A Python project for capturing and analyzing metrics for different LLM providers.

## Usage
1. Create an instance of the `ModelMetrics` class.
2. Add LLM providers using the `add_provider` method.
3. Get metrics for a provider using the `get_metrics` method.
4. Calculate the average of a metric for a provider using the `calculate_average` method.
5. Convert the metrics to JSON using the `to_json` method.

## Testing
Run the tests using `pytest` in the `tests` directory.
